## 2026-03-28 - Optimize `#getCandidates` with single-pass `for` loop
**Learning:** Chained array methods (`.map().filter()`) on hot paths like account selection (`#getCandidates` in `HybridStrategy`) can cause significant performance degradation. Each `.map()` and `.filter()` call iterates over the entire array and allocates new intermediate array objects. This increases execution time and creates unnecessary garbage collection pressure, which is particularly harmful in high-throughput proxies.
**Action:** When evaluating multiple conditions or creating different fallback tiers from a single array, use a single-pass `for` loop. Categorize items into pre-allocated arrays simultaneously, bypassing multiple full-array scans and preventing the creation of temporary intermediate array objects.
